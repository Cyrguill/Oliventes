# SEO launch checklist

## Before public launch

- Add the site to Google Search Console as `https://oliventes.com/`.
- Submit `https://oliventes.com/sitemap.xml` in Search Console.
- Add the site to Bing Webmaster Tools and submit the same sitemap.
- Connect an analytics tool, such as Google Analytics or Plausible.
- If Google Search Console requires an HTML verification tag, add it to the `<head>` of `index.html`.

## After launch

- Inspect the homepage URL in Google Search Console and request indexing.
- Inspect the main service page: `https://oliventes.com/services-coaching-vente-automobile.html`.
- Inspect the five topic pages: appels entrants, prise de rendez-vous, objections prix, relances, closing showroom.
- Monitor indexing, impressions, and search terms weekly for the first month.
- Add new content pages based on real search queries that begin to appear in Search Console.
